# Polynomial Plotter

## Introduction

This project is a ludic app to train my skills in programming and mathematical simulation (easy for now).

It's a simple graph plotter of a polynomial function from degree 0 to 5, in real time with the easiest UI possible.

The goal is to demonstrate my capacity to optimize the backend, frontend, and documentation on a lightweight app to use this type of code next time on a more massive simulation.
